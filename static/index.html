Momentum Scout
시장 판단, 신규 진입 승인, 관찰 후보, 보유 종목 리뷰를 한 화면에서 봅니다.
처음이면 기본값으로 바로 시작해도 됩니다.
평소에는 추천 기본값만 보고 시작해도 됩니다. 바꾸는 값마다 왜 필요한지 아래에 같이 적어뒀습니다.
빠른 시작
KR은 대형주 포함 공격형이 기본 추천입니다. ETF/ETN은 기본 제외하고, 거래대금 상위권과 대형주를 우선 봅니다.
US는 스캔 프로필과 유동성 기준만 먼저 고르면 됩니다. 나머지는 기본값으로 시작해도 충분합니다.
1
스캔 프로필
무엇을 찾고 싶은지 먼저 고르면, 아래 설명과 추천 기본값이 같이 바뀝니다.
KR은 기본 모드가 대형주 포함 공격형으로 고정됩니다.
2
시장과 보유 기간
어느 시장을 볼지, 모멘텀을 며칠 기준으로 볼지 정합니다.
3
기본 필터
평소에는 아래 몇 가지만 보고 시작하면 됩니다. 고급 설정은 정말 바꾸고 싶을 때만 여세요.
KR 기본값은 ETF/ETN 제외 + 거래대금 상위권 우선 + 대형주 누락 최소화 기준으로 저장됩니다.
Preset: KR 대형주 포함 공격형
4
직접 확인할 종목
종목명이나 코드를 그대로 넣으면 최대한 자연스럽게 해석합니다.
입력 팁
KR은 삼성전자, 한화오션처럼 종목명으로 적어도 자동 해석을 시도합니다.
US는 TSLA, NVDA처럼 티커 입력이 가장 정확합니다.
US 후보 소스 고급 설정
보통은 건드릴 필요 없습니다. US 스캔 프로필은 종목 성향을 정하고, 여기서는 후보를 어디서 먼저 가져올지만 정합니다.
직접 입력 우선
직접 입력 종목이 있으면 그 종목을 먼저 보고, 없을 때만 위 후보 소스를 씁니다.
공개 데이터 기반 참고 도구입니다. 실제 주문 전에는 반드시 직접 한 번 더 확인하세요.
스캔 결과
시장 판단을 먼저 보고, 신규 진입 가능 / 관찰만 / 비추천을 나눠서 확인합니다.
선택 가이드
스캔 후 자동 선택과 다음 행동을 여기서 안내합니다.
AI 프롬프트 / 원본 데이터
무엇을 복사할지 먼저 고르고, 어떤 용도인지 확인하세요.
버튼 차이
AI 판단용 프롬프트는 ChatGPT 같은 모델에 바로 붙여넣기 좋은 텍스트입니다.
원본 데이터(JSON)는 숫자와 구조를 그대로 확인하거나 검증할 때 보는 자료입니다.
현재 선택: 없음
신규 진입 승인 후보인지, 관찰 후보인지, 보유 종목 리뷰를 같이 넣는지에 따라 프롬프트가 달라집니다.
원하는 종목을 먼저 체크한 뒤 버튼을 누르세요.
Intraday Desk
실시간 데이트레이딩 보조 데스크
카드형 추천보다 상태 전이와 타이밍을 먼저 보여주는 인트라데이 작업공간입니다.

라이브 레이더
시장 상태, 세션, 신규 진입 허용 여부가 여기에 표시됩니다.
Live Radar
지금 볼 만한 후보와 상태 전이를 함께 봅니다. 핵심은 “지금 사라”보다 “언제 사야 하는지”입니다.
Snapshot
추천이 생성된 순간의 상태와 트리거 플랜을 따로 저장해 나중에 결과와 비교합니다.
Active Trades
이미 진입했거나 곧 진입 가능한 후보를 따로 모아, 손절/익절/남은 리스크를 빠르게 봅니다.
Journal
추천 순간 스냅샷과 별도로, 실제 진입/청산 결과를 기록해 성과를 누적합니다.
추천 결과 기록

Stats
세션별, 셋업별, 결과별 누적 기록으로 어떤 상황에서 잘 맞는지 보기 위한 통계 영역입니다.
Broker Setup
한국투자 API 웹소켓 어댑터가 붙을 자리를 먼저 만들었습니다. 지금은 설정 흐름과 피드 계획을 연결해둔 상태입니다.

권장 방식입니다. KIS_APP_KEY / KIS_APP_SECRET / KIS_ACCOUNT_NO 환경변수를 읽습니다.
